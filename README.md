## Como publicar no GitHub Pages
1. Entre no GitHub.
2. Crie um novo repositório chamado `golden-paws-care`.
3. Faça upload dos arquivos `index.html`, `style.css` e `logo.svg`.
4. Clique em `Commit changes`.
5. Vá em `Settings`.
6. Clique em `Pages`.
7. Em `Build and deployment`, escolha:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
8. Clique em `Save`.

O link ficará parecido com:
https://SEU-USUARIO.github.io/golden-paws-care/
